    Our Mission

    Girl Develop It is a nonprofit organization that exists to provide affordable and accessible programs to women who want to learn web and software development through mentorship and hands-on instruction.

    Our Vision

    Our vision is to create a network of empowered women who feel confident in their abilities to code and build beautiful web and mobile applications. By teaching women around the world from diverse backgrounds to learn software development, we can help women improve their careers and confidence in their everyday lives.

    Our Values

    We are committed to making sure women of all ages, races, education levels, income, and upbringing can build confidence in their skill set to develop web and mobile applications. Our goal is to provide powerful hands-on programs to women seeking professional help in software development and create basic to advanced web and mobile applications

    Our Events

    We have several kinds of events: networking, seminars and core curriculum. The networking meetings and seminars are free and open to anyone. The core curriculum classes have low fees and are for those who interested in getting heads down into some aspect of programming.

    GDI TriCities Core Curriculum

    Intro to Programming: For students with little or no knowledge of multiple languages, the course provides an overview of several languages and how to apply basic techniques.

    Intro to HTML/CSS: The language of building websites. An introductory HTML and CSS course guiding students through basic HTML and CSS building for websites. The course breaks down the history, basics, functions, and applicability of the language. This is typically a 2-day weekend or 4-week course (8 hours total).

    JavaScript/jQuery: The language of interactivity on the web. An introductory course to JavaScript/jQuery teaches the foundation of basic JavaScript, to the jQuery library, dynamic HTML, JSON and APIs. Basic knowledge of HTML is needed on part of the student. This is typically a 4-week course (8 hours total).

    Git/GitHub: The git course teaches student version control management in an open-source environment that encompasses small-large scale change in documents, computer programs, and other collections of information.

    Intermediate HTML5/CSS3: An intermediate class for students with prior HTML and CSS knowledge, this course teaches the fundamentals of HTML5 and CSS3. Students will learn information on advanced selectors, web fonts for typography, graphical enhancements, and more. This is typically a 4-week course (8 hours total).

    UX: The course teaches the fundamentals of web experience and what defines an excellent user experience through lecture, discussion and hands-on exercises. No coding or design experience is required. (Workshop)

    API: Taught at the advanced level, this course teaches JavaScript-based APIs actually work behind the scenes and firsthand experience using API’s. The course introduces students to various API’s to enhance websites.
